Catchhub Landing Page

A modern and responsive landing page built with HTML5 and CSS3, inspired by the provided Catchhub design. This project focuses on recreating the UI with attention to layout, spacing, typography, colors, and visual consistency.

✨ Features

- Fully Responsive Design
- Clean and Modern User Interface
- Hero Section with Download Buttons
- Channels Showcase Section
- Messages Section
- Future Features Section
- Footer with Contact Information
- Optimized Image Assets
- Pixel-Perfect Frontend Implementation

- CSS3
